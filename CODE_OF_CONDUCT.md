# Code of Conduct

## Our standard

We want Thrust to be an open, constructive, and technically rigorous project.

Examples of positive behavior:
- being respectful in disagreement
- focusing criticism on ideas, not people
- giving actionable feedback
- assuming good intent
- helping improve docs, tests, and examples

Examples of unacceptable behavior:
- harassment or personal attacks
- insulting or dismissive language
- bad-faith arguments
- deliberately disruptive behavior
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

## Scope

This applies to issues, pull requests, discussions, and other project spaces.

## Reporting

If there is a problem, contact the project maintainers privately.
